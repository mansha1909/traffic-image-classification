🚦 Traffic Scene Classification using Feature Engineering
📌 Project Overview

This project focuses on Traffic Scene Classification using Feature Engineering Techniques on the TrafficNet dataset.
The system analyzes road traffic images and classifies them into categories such as:

🚗 Dense Traffic
🛣️ Sparse Traffic
🚨 Accident
🔥 Fire

Unlike deep learning-based black-box systems, this project emphasizes:

Image preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Feature Transformation
Feature Scaling
Outlier Detection
PCA Analysis
Rule-based analysis
